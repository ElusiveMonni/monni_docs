# Timestamp

A command for quickly converting times to UNIX Timestamps.

:::slash
**Timestamp** is a [slash command](/commands/info/slash/)!
:::

## Arguments

- 1 **time:**  
    The time you wish to get a timestamp for
	    - Ex: `5 Days`, `4 Weeks`
